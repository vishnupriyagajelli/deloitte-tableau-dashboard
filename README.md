# deloitte-tableau-dashboard
Factory machine downtime analysis using Tableau - Deloitte Forage Simulation

Daikibo Factory Downtime Analysis — Deloitte Forage Simulation
This is a Data Analytics project from the Deloitte Job Simulation on Forage
I analyzed 1,60,704 records from 4 factories across Japan, Germany, and China to find which factory had the most machine downtime

Built an interactive Tableau dashboard that showed:
- Osaka Japan factory had the highest downtime — 480 minutes
- Berlin Germany factory had the lowest — 20 minutes
- Total downtime across all factories — 1,030 minutes

Tool used: Tableau
Completed: March 2026
